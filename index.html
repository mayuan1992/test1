1.memorizing

Based on the timing and quantity the owner feeds the pet in the initial weeks,the system automatically memorize  
the routine and will work in the same calendar.  
Our current idea is to set 2 working modes corresponding to memory and routine condition,of course the user can reset  
the calendar by restarting the memory mode.

2.notification

a.Via measuring the volume the pet eats per time,the system informs the user if he/she should provide more/less food each meal;  
b.The system gets aware when the pet finishes eating and would inform the user his/her pet might need to be brought to the bathroom or outside;  
c.If the food remains the same for a certain time,the system informs the user it's possible that the pet is ill.

3.self-study

This mode is in parallel with the memorizing mode.   
Once self-study mode is set,the system asks an initial feeding value,and adjusts the value according to the pet's behavior(same as 2.a).
